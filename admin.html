Aproape gata

Panoul de administrare are nevoie de o bază de date Supabase (gratuită). Durează ~5 minute, o singură dată.

  1. Creează cont gratuit pe supabase.com și apasă „New project".
  2. În SQL Editor, lipește tot conținutul fișierului supabase-setup.sql și apasă Run.
  3. Mergi la Project Settings → API și copiază Project URL și cheia anon public.
  4. Deschide config.js și pune-le la supabaseUrl și supabaseKey.
  5. Mergi la Authentication → Users → Add user și creează-ți contul tău (email + parolă).
  6. Reîncarcă această pagină și loghează-te.
